# E-commerce Sales & Demand Forecasting

## Project Overview
End-to-end data analytics project using the UCI Online Retail II dataset.
Covers data cleaning, exploratory analysis, cloud SQL analytics, and ML-based demand forecasting.

## Tech Stack
- **Python** (Pandas, NumPy, Scikit-learn, XGBoost, Prophet)
- **SQL** (Google BigQuery)
- **Cloud** (GCP Cloud Storage + BigQuery)
- **Visualization** (Matplotlib, Seaborn, Plotly, Power BI)
- **Version Control** (Git + GitHub)

## Project Phases
1. Environment Setup & Project Scaffolding
2. Data Loading, Cleaning & Quality Checks
3. Exploratory Data Analysis (EDA)
4. Cloud Setup & SQL Analytics on BigQuery
5. Demand Forecasting ML Model
6. Power BI Dashboard
7. Storytelling & Portfolio Publishing

## Dataset
[UCI Online Retail II Dataset](https://archive.ics.uci.edu/dataset/502/online+retail+ii)
